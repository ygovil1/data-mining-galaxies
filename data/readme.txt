Please read this file first.

The calculated halo data is encoded in simple ASCII format in this folder. Each value is separated by a space. Below is a list, in order, of the quantities present, along with their units.

0= new index 
1= run index 
2= x (kpc)
3= y (kpc)
4= z (kpc)
5= new_mass (Msun)
6= new_rad (kpc)
7= isSatellite 
8= Mgas (Msun)
9= Mstar (Msun)
10= ave age (Myr)
11= proximity indicator 
12= stellar effective radius (kpc)
13= stellar formation rate (Msun/Myr)
